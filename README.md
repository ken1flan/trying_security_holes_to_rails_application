# Making security holes to rails application

## はじめに
これは「Railsアプリケーションにがんばってセキュリティホールをあける(仮)」の執筆リポジトリです。

## セットアップ

```bash
$ bundle install
```

## epubのビルド

 ```bash
 $ bundle exec rake epub
 ```
